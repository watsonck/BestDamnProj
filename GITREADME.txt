helpful commands:

to add a file to be pushed:

	git add <filename>

to commit the changes:

	git commit -m <a message to say what youre adding>

push it to the repository:

	git push

to download changes:

	git pull


to change the branch:

	git checkout <branch name>

to clone the repository:

-first go to the directory you would like save it in
- type the following

	git clone https://github.com/YOURUSERNAME/BestDamnProj.git
